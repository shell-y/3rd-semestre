array.slice(inicio, fim);

inicio → índice onde começa a cópia.
fim → índice onde termina (não incluído).
Se algum valor for negativo, ele é contado de trás para frente.

No caso de slice(-3):
O -3 significa “comece 3 posições antes do final”.
Como não passamos o segundo parâmetro (fim), ele vai até o fim do array.
Assim, retorna os 3 últimos elementos.

{11/09/2025} REACT

Foi criado para otimizar o desempenho nas renderizações do Facebook

Framework ~> você precisa codar se adaptando ao ambiente do framework
Biblioteca ~> você pode importá-la no seu código

Melhorou o desempenho na renderização pois criou o ~*Virtual DOM*~
que cria uma DOM virtual em memória e transmite para a DOM "real" somente depois de finalizada, ou seja, renderiza uma só vez.

Diffing ~> "variável" onde guarda as alterações
Reconciliation ~> faz uma comparação entre o diffing e o antigo código, altera somente o que foi modificado e aplica na DOM "real"

Vite ~> ferramenta para criar a estrutura do projeto em react

props (padrão no mercado para a palavra 'propriedades') ~> importar características externas. Basicamente um parâmetro.

CICLO DE VIDA ~> etapas para um componente ser renderizado/atualizado
Consiste em 3 etapas:
	I. Montagem
	II. Atualização
	III. Desmontagem

STATE ~> Armazena informações do componente. Uma variável que persiste entra as atualizações.
É usado quando precisamos atualizar dados dinamicamente que serão refletidos na interface.

Quando chamamos uma função com parênteses(), ela é chamada automaticamente ao carregar a página. Para ela ser iniciada somente quando chamada, mencionar a função SEM os parênteses.
